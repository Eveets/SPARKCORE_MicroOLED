SparkCore-MicroOLED
==================

An adaptation of Sparkfun's library for the MircoOLED breakout to use with SparkCore
https://www.sparkfun.com/products/13003

Hookup
---

MicroOLED ----------- SparkCore

GND			GND

3V3			3V3*

D1/SDI			A5

D0/SCK			A3

D2/SDO			None

D/C			D4 (Any digital pin will do)

RST			D5 (Any digital pin will do)

CS 			A2
